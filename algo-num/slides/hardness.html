RSA Cryptosystem

  • Invented by Ron Rivest, Adi Shamir and Leonard Adleman in 1979
    Independently by Clifford Cocks in 1973 (declassified in 1997)

  • Asymmetric encryption system

    • Receiver generates public and private key
      Publishes public key

    • Anyone can encrypt using public key

    • Only receiver can decrypt (using private key)

    • No key sharing problems!

  • Widely used even today

  • Encryption of x∈Zn∗x \in \mulInt{n}x∈Zn∗​: ciphertext y=xey = x^ey=xe

  • To encrypt large text:

    • Pick random key kkk-bit key: s∈Zn∗s \in \mulInt{n}s∈Zn∗​

      Encrypt actual message using symmetric encryption (eg. AES) with key sss.
